# PostMan发送json格式的post请求

1、headers项

添加一项: Content-Type

取值为: application/json

![image-20211224233342482](PostMan发送json格式的post请求.assets/image-20211224233342482.png)

2、body项

形式选择：raw

格式选择：JSON
